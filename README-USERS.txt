Fast Search - Quick Start Guide for Users
==========================================

Thank you for downloading Fast Search!

WHAT IS FAST SEARCH?
--------------------
Fast Search is a lightning-fast desktop search tool that lets you find
any file on your computer in under 1 millisecond. Press Ctrl+Space from
anywhere to search!

INSTALLATION (One-Time Setup)
------------------------------
1. Make sure Python 3.10 or higher is installed
   Download from: https://www.python.org/downloads/
   IMPORTANT: Check "Add Python to PATH" during installation!

2. Double-click "INSTALL.bat" in this folder

3. Wait for installation to complete (2-3 minutes)

4. Done! A shortcut will be created on your desktop

HOW TO USE
----------
1. Double-click "Fast Search" shortcut on your desktop
   OR run "launch-desktop-app.bat"

2. The app will start in your system tray (bottom-right corner)

3. Press Ctrl+Space from ANYWHERE to open search

4. Type your search query and press Enter to open files

FIRST TIME SETUP
----------------
After launching for the first time:

1. Click the Settings button (⚙️ icon)

2. Add directories you want to search:
   - Documents
   - Downloads
   - Desktop
   - Or any custom folder

3. Click "Add Directory" for each folder

4. (Optional) Enable "Start with Windows" to auto-start

KEYBOARD SHORTCUTS
------------------
Ctrl+Space    - Open search (works from any app!)
↑ ↓           - Navigate through results
Enter         - Open selected file
Ctrl+Enter    - Show file in folder
Esc           - Clear search

TROUBLESHOOTING
---------------
Q: Ctrl+Space doesn't work
A: Try running as administrator, or check if another app uses Ctrl+Space

Q: No search results
A: Make sure you've added directories in Settings

Q: Installation failed
A: Make sure Python is installed and added to PATH

Q: How do I uninstall?
A: Just delete this folder and the desktop shortcut
